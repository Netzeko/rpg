Version dev : 
add member : ajoute un personnage à l'équipe
switch invisibility : active/désactive le mode invisible (les ennemies ne peuvent pas vous attaquer)
add potion : ajoute une petite potion à votre inventaire
add armor set : ajoute quelques pièces d'équipement à votre inventaire
resurrect member : réssuscite le membre de l'équipe ciblé

Controles : 
- clic sur l'image d'un personnage ou d'un ennemi pour le cibler
- bouton select à coté d'un personnage pour sélectionner ce personnage
- touches 1-6 (pavé numérique) : sélection les personnages de l'équipe
- touche 7 (pavé numérique) : affiche le panneau de caractéristiques du personnage sélectionné
- touche 8 (pavé numérique) : affiche la fenêtre d'équipement du personnage sélectionné
- touche 9 (pavé numérique) : affiche la fenêtre de directives du personnage sélectionné
- glisser/déplacer un objet : déplace un objet d'une zone à une autre (permet d'équiper/de mettre en raccourci/d'acheter/de vendre/de placer dans l'inventaire/d'utilisesr sur une serrure)
- clic droit sur un objet : équipe/déséquipe ou place en raccourci/enlève du raccourci un objet
- double clic sur un objet : utilise l'objet
- flèches directionnelles : se déplacer dans le donjon
- barre d'espace : interagir (équivalent du bouton central bas)

A savoir :
- Le combat est engagé avec les ennemis dès que vous êtes sur la même case
- Augmenter une caractéristique de 1 coute autant de points de la valeur actuelle de la caractéristique
